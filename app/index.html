--- title: Philanthrodex - The Grants Data Index layout: main ---

This is the web version of the grants and proposals index, derived from Philanthrobotics - the grants commons knowledge graph and analytics -- maintained at this PubPub. This index is an index of indexes, and contains grant proposals that have been submitted to various organizations, availabilities in open contests, and tools to analyze grants data. To edit, you can submit changes to our Airtable, or by making a pull request to our Github repository. You can use the search bar to explore the datasets, or browse the full list directly. More variations coming soon!
